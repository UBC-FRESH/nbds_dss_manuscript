# LLM Agent Guardrails — Sustainable Futures Transfer Packaging

## Purpose
This repo is being repackaged for submission to **Sustainable Futures** (Elsevier).
Your job is to get the manuscript and submission artifacts compliant quickly and cleanly.

## Primary Sources
- **Author instructions:** `reference/Guide for authors - Sustainable Futures - ISSN 2666-1888 | ScienceDirect.com by Elsevier.pdf`
- **Tracking docs:** `roadmap.md`, `submission-readiness-checklist.md`

## Working Rules (must follow)
1. **Do not change scientific content** (results, methods, claims, numbers) unless explicitly asked.
2. **Keep edits minimal and traceable.** Every change should map to a specific SF requirement.
3. **Use the checklist.** Update `submission-readiness-checklist.md` statuses as you complete items.
4. **Keep formatting stable.** Avoid reflowing paragraphs or changing LaTeX structure without need.
5. **File hygiene matters.** Use ASCII filenames, avoid spaces, and update paths accordingly.
6. **Ask before big moves.** If a change might alter meaning, scope, or structure, ask first.
7. **No destructive git actions.** Never reset or overwrite unrelated work.

## Expected Workflow
1. **Scan requirements** in the SF guide for any relevant constraints.
2. **Plan tasks** in `roadmap.md`, then execute in order.
3. **Update manuscript** (`manuscript.tex`) for SF-specific requirements:
   - Journal name, acknowledgements placement, AI disclosure title/wording, funding role statement, data policy wording.
4. **Align submission assets**:
   - `highlights.txt`, graphical abstract, figure files (separate + naming convention).
5. **Build and QA**:
   - Recompile PDF, spot-check references, appendix numbering, and figure/table citations.
6. **Package** the submission sources and assets.

## Deliverables
- Updated `manuscript.tex`
- Updated `submission-readiness-checklist.md`
- Updated `roadmap.md`
- Clean submission-ready figure files + highlights + graphical abstract

## When in Doubt
State the uncertainty, point to the exact guideline section, and ask for a decision.
